balance_check¶

Given a year compare the values in fcast.xlsm and ‘<year> Account Balances.tsv’

After a bit of fiddling to get the accounts to match, does a left join and prints out how closely the values match

Functions

compare(year)

reads the values for a year from a saved the Account Balances report generated by Moneydance and matches to those to the values from fcast.xlsm
